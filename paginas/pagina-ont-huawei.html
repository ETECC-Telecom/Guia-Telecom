Configuração Inicial (Primeiro ou após reset)

Agora iremos aprender a como configurar a ONT Huawei de forma manual. O primeiro passo é realizar o Reset, que nesse caso, basta desligar a ONT, e ao ligar, pressione por 15 a 20 segundos o botão reset.

Para se conectar a ONT, você pode acessar diretamente pelo WiFi, com os dados encontrados na na etiqueta atrás da ONT. Se por algum motivo não consiga se conectar ao WiFi, então você precisa configurar um ip local fixo na faixa de IP do Router, por exemplo: 192.168.18.100 (Procedimento esse que já foi ensinada na página inicial de Configuração de Router).

Quando estiver conectado no ativo, você consegue acessar o mesmo pelo IP do Gateway. Agora basta colocar o Login Sensível, e senha Sensível, e caso a senha nao funcione, use o Sensível e a senha que está na etiqueta do ativo.

Agora iremos iniciar as Configurações. Após login, você será direcionado para uma página de configuração inicial do ativo. Evite usar essa função, vamos clicar no botão Exit para seguir o Passo a Passo.

Agora vamos habilitar as portas LAN em Menu Lateral Esquerdo (Engrenagem) -> LAN -> Layer 2/3 Port. Na página que abrir selecione os checkbox de LAN1 a 4 e clique no botão Apply.

Na página do ativo, vamos ao menu lateral esquerdo (engrenagem) -> WAN e vamos clicar no botão NEW.

No formulário que surgir, habilite as opções:

  • Ative o Enable Wan;
  • Encapsulation Mode: Selecione PPPoE;
  • Protocol Type: Selecione IPv4 e IPv6;
  • Wan Mode: Selecione Router WAN;
  • Service Type: Selecione a opção TR069_INTERNET;
  • Enable VLAN: Ative a opção e sete a VLAN em VLAN ID;
  • MRU: Adicione o valor 1492;
  • User Name e Password: Coloque o PPPoE do cliente;
  • Habilite todos os checkboxs de LAN e SSID1 e 5;

Após realizar todos esses ajustes, vamos ao final da página e clicar em Apply. E esse é o básico para configurar esse modelo de ONT manual, o resto para subir acesso é provisionar pela interna e atualizar o MAC.

O restante das configurações, pode ser as mesmas dos tutoriais a baixo.

Alteração de Senha de Acesso

Alterar a senha de Acesso da ONT Huawei é extremamente simples. Basta acessar o ativo, e no menu lateral esquerdo, clicar na engrenagem, ir até a aba System Management -> Account Management. Na janela que se abrir, selecione no formulário o User Name desejado, coloque a nova senha e confirme a nova senha nos campos New Password e Confirm Password:

Configuração de PPPoE

Para realizar a alteração do PPPoE em uma ONT Huawei já configurada é simples. Basta acessar a mesma, navegar até o menu lateral esquerdo e clicar na engrenagem -> WAN e na janela que se abrir selecione a linha de configuração da rede de internet do cliente. No exemplo a baixo temos duas linhas, uma para internet e outra para a telefonia.

Após clicar na linha de conexão, será exibido um formulário, no qual algumas opções podem ser alteradas, e dentre essas opções temos a linha de User Name e Password, onde iremos inserir o novo PPPoE:

Habilitar Acesso Remoto

  • Na página do router, selecione a opção Segurança→ Controle de Acesso de Dispositivo Preciso.
  • Após isso click que no Botão Novo.
  • Na página do router, selecione a opção Security→ Precise Device Acess Control.
  • Após isso click que no Botão New.
  • Na janela que se abrir, vamos preencher os seguintes dados:
  • Type: Aplication;
  • Priority: 1;
  • Port Type: WAN;
  • Port name: All;
  • Source IP Enddress: 177.185.46.0 e 177.185.46.254;
  • Mode: Permit
  • Agora, clique no botão apply, e em seguida, new novamente para uma nova regra. Nesse novo, iremos preencher
  • os parâmetros da seguinte forma:
  • Type: Aplication;
  • Priority: 2;
  • Port Type: WAN;
  • Port name: All;
  • Source IP Enddress: 138.94.84.0 e 138.94.84.254;
  • Mode: Permit
  • Após fazer isso seu acesso remoto estará habilitado nesse dispositivo.

Configuração de WiFi

Agora varemos como configurar o WiFi da ONT Huawei. Existem duas formas de configurar o Wifi desse ativo. A primeira, é só para alterar o nome e senha, e a segunda é para fazer alterações de configurações mais avançadas.

Vamos começar pela configuração de SSID e Senha:

Assim que você acessa a ONT você cai na página de topologia da mesma. Basta clicar no ícone do Wifi, que será exibido para você um formulário tanto para a alteração do SSID e Senha da rede 2.4G quanto 5G. Após alteração basta salvar e esperar o ativo aplicar as configurações:

Alterações Avançadas:

Agora para configurações avançadas da rede WiFi, basta clicar no ícone da engrenagem no menu lateral esquerdo -> WLAN -> e clicar na opção "Advanced Network..." para ter acesso a página de configurações avançadas como: Canal, Domain, mode e entre outras configurações avançadas.

N.T.: Para a Configuração avançada de WiFi, temos ao todo duas páginas para cada rede (2.4G e 5G). Essas configurações são conceitos mais avançados que não serão abordados nessa sessão, mas vale a pena pesquisar!

Acionamento Manual da Telefonia

Antes de qualquer coisa, para uma analise simples da telefonia na ONT, você pode consultar após configuração o numero de telefone facilmente pela página inicial:

Para acionamento manual da Telefonia, vamos primeiro no Menu Lateral Esquerdo (*Engrenagem*) -> WAN e vamos adicionar uma nova regra em New, assim como fizemos na configuração da WAN. Agora mantenha habilitado o WAN.

No formulário que aparecer, vamos configurar da seguinte forma:

  • Enable WAN: Ative a caixa;
  • Service Type: VOIP;
  • Enable VLAN: Habilite a opção e adicione a VLAN ID para o valor 120, ou o valor que foi passado previamente;
  • MTU: Adicione o valor 1500;
  • IP Acquisition Mode: DHCP;

E para finalizar essa etapa clique em "Apply";

E para finalizar vamos no menu lateral novamente -> Voice -> VoIP Basic, e vamos apenas configurar algumas opções:

  • Outbound Proxy Server Address: Esse valor é padrão 138.94.87.116
  • Outbound Proxy Server Port: Normalmente passado para o técnico pela OS;

Já as demais configurações são padrão e já vem habilitado.

Mais a baixo devemos agora apenas preencher o formulário com o numero do cliente em Registration User Name, Habilitar o Enable User, e adicionar o valor 1 em Associated POTS Port:. Já em Authentication User Name: novamente colocamos o numero do cliente e a Password geralmente é passada ao técnico, porém costuma ser a padrão #Etecc111426.

E para finalizar, clique em "Apply".
